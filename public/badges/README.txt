- /badges/first_saver/level-1.png ... level-6.png
- /badges/saving_streak/level-1.png ... level-6.png
- /badges/budget_keeper/level-1.png ... level-6.png

Fallback (optional):
- If level image is missing, app tries /badges/<badge_key>.png
- If still missing, app uses default trophy icon

Notes:
- Preferred size: 256x256 px (square).
- Use PNG for easiest setup with current mapping.
